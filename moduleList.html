Module List

Pazzo

Structor

helpers/

  • 非 UI 用的共用資源，例如協助環境判斷的函式，或是 sass reset。

components/

  • 共用的 UI 模組及 UI 相關函式，模組部分包含 js, pug & React，如果有適合共用的 sass 也可以放這裡。

vendors/

  • 從外部下載後直接被引用的資源。

React Modules

components/

  • global
    • Pagination (分頁選擇器)
    • ImagePlaceholder (圖片 404)
    • Select (下拉選單)
    • Lightbox (顯示 React 模組的光箱)
  • cart
    • ReactMiniCart (由 React 實踐的迷你購物車渲染及相關事件)
  • product
    • Breadcrumb (麵包屑)

JavaScript Modules

helpers/

  • excuteByDom (當指定 dom 存在時執行方法)
  • ignoreByDom (當指定 dom 存在時不執行方法)
  • versionSwitch (不同螢幕寬度時執行不同程式)
  • timer (使方法在伺服器時間區間內被呼叫)
  • preloadimages (預載圖片並回傳 .done() 方法)
  • setDefaultVal (設定預設值)
  • getURLParameter (查詢網址參數內容)
  • getServerTime (取得伺服器時間)
  • objectFilter (使陣列內的物件依照該物件的特性被篩選，回傳篩選後的陣列)

components/

  • global
    • image404 (圖片 404)
    • dialogPopup (ajax 動態載入彈跳光箱)
    • htmlNoTouch (html 加上 .no-touch)
    • noContextMenu (點擊右鍵無效)
    • triggerSelect (綁定 select mousedown 事件)
    • focusInput (有資料的 input 維持 focus 狀態)
    • uiSelect (自訂樣式下拉選單)
    • uiQuantity (數量選擇器)
    • uiTabs (點擊標籤切換顯示區塊)
    • gotop (回到頂端按鈕)
    • accordion (手風琴)
    • hashAccordion (url hash 與手風琴)
    • slideNotice (於指定物件添加/刪除提示文字 HTML 標籤)
    • floatLabel (表單輸入框浮動提示標籤)
    • rigMargin (使用 margin-top 定位 DOM)
    • clickByEnter (按 Enter 時點擊按鈕)
    • formInputOrder (表單中點擊 Tab 及 Enter 事件)
  • template
    • bodyDarken (捲動切換背景色)
  • productList
    • itemFilter (列表篩選工具)
  • product
    • zoombox (圖片列表放大)
